# NetSuite Scheduled Script RESTlet Call
Simple Scheduled script that call NetSuite RESTlet and get JSON data

## Used libraries
- oauth
- crypto-js

## Features
- Dynamic URL Discovery for RESTlet Clients ( using The NetSuite DataCenterUrls REST Service
  )
